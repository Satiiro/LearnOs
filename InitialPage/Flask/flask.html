O que é o Flask?

O Flask é um framework escrito na linguagem de programação Python, ele está na categoria de miniframeworks pois não exige ferramentas ou blblioteca particulares. Ele foi desenvolvido em 2010 por Armin Ronacher, sendo baseado em duas coisas, no kit de ferramenta chamado WSGI e na biblioteca chamada Jinja2, essa sua facilidade pode ajudar principalmente novos desenvolvedores que querem desenvolver aplicativos web mais rapidamente. Ele tem como benefícios:

  • Simplicidade;
  • Soluções fáceis;
  • Rapidez;

Pré-requesitos

Há alguns requesitos minímos para se utilizar a framework Flask, mas sendo esta uma microframework, estes requesitos não chegam a ser extremamente complicados.

  • Um ambiente de programação Python 3, como por exemplo o Repl.it
  • Conhecimento breve sobre programação Python, como loop, funções, entre outros.

Dado estes pré requesitos, já é possível se utilizar do Git para o controlo de versão.
