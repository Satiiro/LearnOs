O que é o PyTest?

PyTest é uma estrutura de teste que permite aos usuários escrever códigos de teste usando a linguagem de programação Python. Ele ajuda você a escrever casos de teste simples e escaláveis para bancos de dados, APIs ou UI. PyTest é usado principalmente para escrever testes para APIs. Ele ajuda a escrever testes desde testes unitários simples até testes funcionais complexos. Ele tem como benefícios:

  • Muito fácil de começar por causa de sua sintaxe simples e fácil;
  • Código aberto;
  • Pode executar testes em paralelo;
  • Detectar testes automaticamente;

Pré-requesitos

Há alguns requesitos minímos para fazer testes com o PyTest:

  • Python 3.6 ou posterior e um ambiente virtual com o Pytest instalado em seu computador.Repl.it
  • Conhecimento breve sobre programação Python, além de familiaridade com ferramentas de linha de comando e execução de scripts.

Dado estes pré requesitos, podemos partir para a instalação do PyTest.

Primeiros passos

Para instalar o PyTest vamos precisar do cmd

Primeiro passo:

Ao abrir o cmd iremos executar o comando "pip install pytest==2.9.1"

Segundo passo:

Agora para confirmarmos que a instalação foi um sucesso digitaremos esse comando "py.test -h"
