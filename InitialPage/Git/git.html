O que é o Git?

O Git é um programa conhecido mundialmente para controle de versão. Criado por Linus Torvalds sendo lançado a 17 anos atrás em 07 de abril de 2005. Ele ficou extremamente conhecido por ter ferramentas diferentes de outros programa para controle de versionamento. Entre eles podemos citar o snapshot, onde ele basicamente tira "fotos" dos arquivos selecionados e salva no servidor. Facilitando a vida do desenvolvedor que vai se preocupar menos em caso de perda de algum arquivo, já que uma cópia estará sempre disponível. Seus benefícios são:

  • Segurança
  • Projetos colaborativos mais facilmente
  • Capacidade de ramificação

Pré-requesitos

O git é um programa, logo ele precisa ser instalado no seu computador. Há alguns requesitos que precisam ser seguidos para a sua utilização correta, sendo eles:

  • Estar instalado no computador, é possível instalar ele através do site;
  • Alguma linguagem de programação instalada no computador, além de uma IDE (Sublime, Visual Studio, Pycharm);
  • Uma conta de

Dado estes pré requesitos já é possível começar a configuração inicial para posteriormente usar o Git!
