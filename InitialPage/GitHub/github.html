O que é o GitHub?

O GitHub é uma plataforma web que pode ser considerada indispensável para qualquer pessoa que desenvolva aplicações, independente para qual plataforma seja. Ele serve tanto para armazenar os códigos dos usuários, para controle das versões, como também para transformar o compartilhamento desses arquivos mais fáceis servindo como uma espécie de "rede social para programadores" neste caso. Ela foi lançada em 2008 e comprada pela Microsoft em 2018. Seus benefícios são:

  • Facilidade de compartilhamento;
  • Fácil entendimento;
  • Funcionalidades extremamente úteis.

Pré-requesitos

Por ser uma plataforma web, o GitHub exige somente uma conta registrada na sua plataforma para poder utilizar!

Primeiros passos

O GitHub é uma plataforma de utilização extremamente simples, e para isso só precisa se utilizar do site oficial GitHub, a partir dele poderemos fazer tudo que se faz necessário para adicionar nossos códigos no site.

Primeiro passo:

O primeiro passo é acessar o site GitHub e criar a sua conta, que exige somente alguns dados.

Segundo passo:

A partir disso já é possível criar um repositório, é o local onde você irá armazenar qualquer coisa que envolva o seu projeto de programação.

Terceiro Passo:

Agora que o seu repositório já está criado, podemos mostrar um programa que será de extrema utilidade e facilitará todo o trabalho, chamado de GitHub Desktop, podendo ser baixado aqui. A sua instalação é bastante simples e não exige nenhum conhecimento prévio.

Através do GitHub Destkop é possível verificar facilitar todo o trabalho para controle das versões, você poderá avaliar de forma mais simplificada todo o processo necessário para colocar novas versões no GitHub! Através dessa ferramenta você poderá copiar o seu repositório através de link, ou adicionar um que está no seu computador em seu GitHub de forma rápida. Qualquer nova alteração feita nos arquivos com ele aberto será exibido, e aí é só commitar através dos botões existentes na interface.

Documentação
